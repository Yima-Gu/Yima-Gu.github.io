TSP

近似算法的思路为先找出原图的一棵最小生成树，然后对这棵树上的结点进行先序遍历得到原图的一个TSP问题的近似解。

上面是在满足三角不等式情况下的近似算法，是2-近似算法。TSP问题的最优解一定是大于最小生成树的边权和的（H回路删掉一条边之后得到的是生成树）。并且对生成树进行完全遍历得到的代价恰好为生成树权值和的两倍，并且由于三角不等式，少经过一个结点不会增加总代价，于是得证。
